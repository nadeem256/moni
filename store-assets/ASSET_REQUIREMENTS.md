## Required Assets for Store Submission

### 1. App Icon
- **Size**: 512x512 pixels
- **Format**: PNG (32-bit)
- **Background**: Not transparent
- **Current**: You have icon.png in assets/images/

### 2. Feature Graphic
- **Size**: 1024x500 pixels  
- **Format**: JPG or PNG (24-bit)
- **Purpose**: Main promotional banner
- **Content**: App name, key features, attractive design

### 3. Screenshots (Required)
**Phone Screenshots:**
- **Minimum**: 2 screenshots
- **Maximum**: 8 screenshots
- **Aspect Ratio**: 16:9 or 9:16
- **Recommended Size**: 1080x1920 or 1920x1080

**Tablet Screenshots (Optional but recommended):**
- **Aspect Ratio**: 16:10 or 10:16
- **Size**: 1200x1920 or 1920x1200

### 4. Screenshots to Take:
1. **Home Screen** - Show balance and overview
2. **Subscriptions List** - Display tracked subscriptions
3. **Add Transaction** - Show the add transaction form
4. **Insights/Analytics** - Display charts and insights
5. **Profile/Premium** - Show premium features
6. **Onboarding** - Welcome screens (optional)

### 5. Video (Optional)
- **Length**: 30 seconds to 2 minutes
- **Format**: MP4, MOV, or AVI
- **Purpose**: App preview/demo

## Creating Screenshots

### Method 1: From Expo Go
1. Run app in Expo Go on your phone
2. Navigate to each screen
3. Take screenshots using phone's screenshot function
4. Transfer to computer

### Method 2: From Android Emulator
1. Run Android emulator
2. Install Expo Go in emulator
3. Run your app
4. Take screenshots from emulator

### Method 3: Using Expo Development Build
1. Create development build: `eas build --platform android --profile development`
2. Install on device
3. Take screenshots

## Tips for Great Screenshots:
1. **Show real content** - Use actual app data, not placeholder text
2. **Highlight key features** - Focus on what makes your app special
3. **Use consistent styling** - Match your app's design language
4. **Include captions** - Add text overlays explaining features
5. **Show the flow** - Screenshots should tell a story
6. **High quality** - Use high-resolution, crisp images
